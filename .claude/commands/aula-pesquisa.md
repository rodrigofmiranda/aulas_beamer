# /aula-pesquisa
## Uso: `/aula-pesquisa N`
## Exemplo: `/aula-pesquisa 7`

---

Este arquivo é um **wrapper fino** para evitar divergência entre o fluxo do Claude e o fluxo canônico do projeto.

## FONTE CANÔNICA DESTA ETAPA

Leia e siga integralmente:

- `CLAUDE.md`
- `aula-pesquisa.md`
- `memoria_projeto.md` (quando existir)
- `fontes_e_precedencia.md` (quando existir)

## INSTRUÇÕES

1. Execute o fluxo completo definido em `aula-pesquisa.md` para a Aula **{{N}}**.
2. Consulte os arquivos adicionais pedidos pelo fluxo canônico, sem reinventar uma versão paralela.
3. Se houver qualquer conflito entre este wrapper e os arquivos canônicos, **vale o arquivo canônico**.
4. Não escreva slides ainda; finalize em **"aguardando aprovação"**.
5. Resposta final da aula em espanhol rioplatense/uruguayo.

## REGRA DE MANUTENÇÃO

Novas regras desta etapa devem ser adicionadas em `aula-pesquisa.md`, não aqui.
