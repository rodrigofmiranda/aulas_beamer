# /aula-escrita
## Uso: `/aula-escrita N`
## Exemplo: `/aula-escrita 7`

---

Este arquivo é um **wrapper fino** para evitar divergência entre o fluxo do Claude e o fluxo canônico do projeto.

## FONTE CANÔNICA DESTA ETAPA

Leia e siga integralmente:

- `CLAUDE.md`
- `aula-escrita.md`
- `layout.tex`
- `docs/regras_pedagogicas.md`
- `docs/regras_visuais.md`
- `docs/regras_tecnicas_latex.md`

## INSTRUÇÕES

1. Execute o fluxo completo definido em `aula-escrita.md` para a Aula **{{N}}**.
2. Consulte as aulas anteriores e os arquivos auxiliares pedidos pelo fluxo canônico.
3. Se houver qualquer conflito entre este wrapper e os arquivos canônicos, **vale o arquivo canônico**.
4. Escreva o `.tex` completo da aula.
5. Não compile nesta etapa; finalize em **"aguardando auditoria"**.
6. Conteúdo dos slides em espanhol rioplatense/uruguayo.

## REGRA DE MANUTENÇÃO

Novas regras desta etapa devem ser adicionadas em `aula-escrita.md`, não aqui.
